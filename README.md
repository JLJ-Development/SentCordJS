# SentCordJS
 SentCordJS is an api wrapper for sentcord to help you post your server/shard stats easily.
 https://sentcord.com

 var key = 'uihfiluwhfiuwhfiusdhfauidfhsdiufhasldufhsdilfuhsdfiulhffjhf'
